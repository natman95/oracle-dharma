Data Flow — การไหลของข้อมูล

Sales Flow (ขาย)

CRM → Quotation → Sales Order → Delivery/Inventory → Invoice → AR → GL → Financial Statements

Purchase Flow (ซื้อ)

Purchase Request → PO → Goods Receipt → QC Inspection → Inventory → Vendor Invoice → AP → GL

Manufacturing Flow (ผลิต)

BOM → Work Order → Material Issue (Inventory) → Production → QC → Finished Goods (Inventory) → Costing → GL

Tax Flow (ภาษี)

Tax Invoice → VAT (ภ.พ.30) → e-Filing
Payment → WHT (50ทวิ) → PND1/3/53 → e-Filing
Payroll → SSO (ส.ปช.) → ประกันสังคม
