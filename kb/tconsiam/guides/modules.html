🏗️ System Architecture

Backend Stack

  • • Bun — JavaScript runtime with native SQLite
  • • Hono — Lightweight web framework
  • • Drizzle ORM — Type-safe SQL builder
  • • bun:sqlite — In-process SQLite
  • • JWT — Token authentication (jsonwebtoken)
  • • bcrypt — Password hashing (Bun.password)

Frontend Stack

  • • React 19 — UI framework
  • • Tailwind CSS 4 — Utility CSS
  • • Zustand — State management
  • • Recharts — Data visualization
  • • lucide-react — Icon set
  • • Vite — Build tool

Infrastructure

  • • nginx — Reverse proxy (port 8890)
  • • systemd — API service management
  • • SQLite WAL — Database (write-ahead logging)
  • • Static files — /opt/tconsiam-dashboard/
  • • API port — 3457 (internal)
  • • UI port — 5173 (dev)

API Response Pattern

{ products: [...] } // Named key wrapping { inspection: {...}, defects: [...] } // Multiple roots as needed { error: "message" } // Errors use 'error' key
