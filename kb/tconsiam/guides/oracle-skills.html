What is arra-oracle-skills-cli?

arra-oracle-skills-cli is an npm package that installs Oracle skills — reusable, AI-first workflow commands — into Claude Code, Cursor, OpenCode, GitHub Copilot, and 14+ other AI coding agents.

Skills extend your AI agent with persistent memory, session awareness, cross-repo pattern discovery, and collaboration features. Install once globally, use everywhere.

Installation & Updates

Install for Claude Code (Standard Profile)

npx arra-oracle-skills@3.6.1 install -g -y --agent claude-code

Install Full Profile (All 26 Skills)

npx arra-oracle-skills@3.6.1 install -g -y -p full --agent claude-code

Install Lab Profile (Full + Experimental)

npx arra-oracle-skills@3.6.1 install -g -y -p lab --agent claude-code

Install Specific Skills

npx arra-oracle-skills@3.6.1 install -g -y -s recap rrr trace --agent claude-code

Install for Multiple Agents

npx arra-oracle-skills@3.6.1 install -g -y --agent claude-code codex opencode cursor
Auto-Update: Use the /soul-sync skill to check and update to the latest version anytime.

Skill Profiles

Three profiles let you choose the right level of functionality. Switch anytime with /go standard, /go full, or /go lab.

standard (16 skills) — Default
about-oracle, awaken, contacts, dig, forward, go, inbox, learn, oracle-family-scan, oracle-soul-sync-update, recap, rrr, standup, talk-to, trace, xray
full (26 skills) — All Features
Includes standard + auto-retrospective, contacts-advanced, create-shortcut, dream, feel, philosophy, resonance, schedule, where-we-are, who-are-you
lab (26 skills) — Full + Experimental
Identical to full profile. Experimental features flagged in skill descriptions.

Creating Custom Skills

Use /create-shortcut to build local skills as quick /slash-commands without publishing to npm.

Create a Custom Skill

/create-shortcut my-skill "Do something cool"

This creates a real executable skill in .claude/skills/ that behaves just like built-in skills. Share the code with teammates or keep it private.

Why use /create-shortcut? Faster than npm publishing. Perfect for team-specific workflows, internal tools, and experiments.

CLI Commands

Command Description
install [options] Install skills to your agent (default: standard profile)
uninstall [options] Remove installed skills
select [options] Interactive skill picker — choose skills to install
list [options] Show all installed skills
profiles [name] List available profiles (standard, full, lab)
agents List 18 supported AI agents
about Show version and status information
