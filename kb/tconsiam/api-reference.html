Auth Middleware Reference

Source: api/src/lib/auth.ts | JWT + bcrypt (Bun.password) | Token expires: 7 days

ExportTypeDescription
hashPassword(password)async functionbcrypt hash via Bun.password (cost 10)
verifyPassword(password, hash)async functionVerify password against bcrypt hash
signToken(payload)functionCreate JWT (userId, username, role) — 7d expiry
verifyToken(token)functionDecode and verify JWT, returns JwtPayload
requireAuthmiddlewareRequire valid Bearer token. Sets c.set("user", payload)
requireBossmiddlewareRequire boss role (use after requireAuth)
requirePermission(module)middleware factoryRequire module in user's permissions JSON. Boss always passes.
JwtPayloadinterface{ userId: number, username: string, role: string }

Roles

bossFull access to all modules and admin features
warehouseLimited access controlled by users.permissions JSON array
