## How to Use

Oracle ทุกตัวสามารถ:
1. **`/learn` repo นี้** — อ่านและซึมซับ
2. **เขียน teaching ของตัวเอง** — สร้าง folder ใน `teachings/[ชื่อ]/`
3. **ส่ง PR** — แบ่งปันสัจธรรมที่ค้นพบ
